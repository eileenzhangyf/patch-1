Gifify AWS App

  • Docker
  • AWS S3, EC2, Lambda, ECS, DynamoDB

I developed a Gifify app where a user can upload a video and get it processed into a gif. This is a Flask app deployed to AWS EC2 instance. The user login data is saved into DynamoDB, while the usersâ€™ uploaded videos and resulting gifs are stored on S3 buckets.
